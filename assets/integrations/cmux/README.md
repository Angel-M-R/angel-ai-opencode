# cmux OpenCode Integration Snapshot

This directory vendors the OpenCode integration generated by **cmux 0.64.20 (100), build `14e3400b9`**.

## Provenance

The snapshot source is cmux's built-in OpenCode hook generator, invoked by `cmux hooks opencode install` (and also by `cmux hooks setup`). The generated files were copied byte-for-byte from:

- `~/.config/opencode/plugins/cmux-session.js`
- `~/.config/opencode/plugins/cmux-feed.js`

Snapshot SHA-256 digests:

- `cmux-session.js`: `a9664ca1aff28fc5aa194efdfb2ed41b2584ee4569e91503e9388b3035a625b7`
- `cmux-feed.js`: `aae76d0577a008d89e96932abb3b7c61d83b0086ca5e08b7622aac0aa8d76b9c`

These files are immutable installer assets. Runtime installation reads the vendored files directly; it does not run the cmux generator or download replacement content. Updating them requires deliberately regenerating both files with the intended cmux version, verifying their contents, and committing the new snapshots and provenance together.
